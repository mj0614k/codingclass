문자열 변경 메서드 : replace() / replaceAll()

replace() / replaceAll() 메서드는 선택한 문자열을 변경하고, 변경된 문자열로 반환합니다.

replace() / replaceAll() 메서드

replace() / replaceAll() 메서드는 어떤 패턴에 일치하는 일부 또는 모든 부분이 교체된 새로운 문자열을 반환합니다. 그 패턴은 문자열이나 정규식(RegExp)이 될 수 있으며, 교체 문자열은 문자열이나 모든 매치에 대해서 호출된 함수일 수 있습니다.


1. "문자열".replace("찾을 문자열", "변경할 문자열")

const str1 = "javascript reference"
const currentStr1 = str1.replace("javascript", "자바스크립트");
const currentStr2 = str1.replace("j", "J");
const currentStr3 = str1.replace("e", "E");
결과보기
자바스크립트 reference
Javascript reference
javascript rEference

currentStr3 변수의 결괏값을 살펴보면, 가장 처음에 나오는 e만 대문자 E로 변환된 사실을 알 수 있습니다. 모든 e를 대문자 E로 변환하려면 어떻게 해야 할까요?


2. "문자열".replace(정규식, "변경할 문자열")

이러한 경우, 정규식 표현을 이용하면 됩니다. 자바스크립트 : 정규식 표현 게시글에서 정규식 표현의 주요 패턴을 볼 수 있습니다. 

const str1 = "javascript reference"
const currentStr4 = str4.replace(/e/g, "E");
결과보기
javascript rEfErEncE

위 예제에서 정규식 표현 중 하나인 g는 Global의 의미로, 대상 문자열 내에 있는 모든 패턴들을 검색하는 것을 의미합니다. 정규식 표현을 사용하는 것 외에도, 손쉽게 replaceAll() 메서드를 사용할 수도 있습니다.


3. replaceAll() 메서드

const str1 = "javascript reference"
const currentStr5 = str1.replaceAll("e", "E");
결과보기
javascript rEfErEncE

replace() 메서드 적용하기

다음과 같이, 다양한 방법으로 정규식 표현을 활용하여 replace() 메서드를 응용할 수 있습니다. 

const str3 = "010-2000-1000";
const currentStr6 = str3.replaceAll("-", "");
const currentStr7 = str3.replace(/-/g, " ");
const currentStr8 = str3.replace(/[1-9]/g, "★");
