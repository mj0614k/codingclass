일러스트레이터 : 토끼 그림 그리기

오늘은 일러스트레이터를 활용하여 그림을 그리는 방법을 알아보도록 하겠습니다.

스케치 작업

다음과 같은 토끼 그림을 그려볼 예정입니다.

일러스트레이터_01_study_01

펜 툴(Pen tool, 단축키 P)로 선을 그어주며 스케치 작업을 먼저 해 주었습니다. 선 두께는 우선 1px로 작업하고, Shift + W을 사용하여 세세하게 조절해 주었습니다.

일러스트레이터_01_study_02

채색 작업

스케치 작업을 마무리했다면, 채색을 해줍니다. 채색 작업을 하기 전, 메뉴의 Object에서 Expand Appearance 작업을 먼저 해 줍니다. 이후, 메뉴의 Object에서 Live Paint를 만들어 줍니다. Live Paint를 사용하면 단축키 K와 Alt(스포이트)를 활용하여 손쉽게 채색 작업을 할 수 있습니다. 다음과 같이 채색을 마쳤습니다.

일러스트레이터_01_study_03

Tip!
상단 바에서 Recolor Artwork를 클릭하면 색상 조합을 보다 손쉽게 할 수 있습니다. 일러스트레이터_01_study_04

명암 작업

마지막으로, 명암 작업을 할 차례입니다. 펜 툴을 다시 선택하여, 배경색과 테두리색을 제거한 뒤 명암을 넣을 부분에 선을 그어 줍니다. 명암 채색을 할 때는 다시 한번 Live Paint를 만들어 주어 작업해 줍니다.

일러스트레이터_01_study_05

Recolor Artwork를 활용하여 토끼 대가족을 만들었습니다.
