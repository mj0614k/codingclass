문자열 검색 메서드 : charAt()

charAt() 메서드는 문자열에서 특정 인덱스에 위치하는 값을 반환합니다.

charAt() 메서드

"문자열".charAt(숫자)

const currentStr1 = str1.charAt();
const currentStr2 = str1.charAt("0");
const currentStr3 = str1.charAt("1");
const currentStr4 = str1.charAt("2");
