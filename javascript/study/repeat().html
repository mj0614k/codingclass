문자열 변경 메서드 : repeat()

repeat() 메서드는 문자열을 복사하여, 복사한 새로운 문자열을 반환합니다. 즉, 이름 그대로, 선택한 문자열을 반복합니다.

repeat() 메서드

다음과 같이, count 안에 문자열을 반복할 횟수(0과 양의 무한대 사이의 정수)를 작성합니다.

str.repeat(count); 


const str1 = "javascript";
const currentStr1 = str1.repeat(0);
const currentStr2 = str1.repeat(1);
const currentStr3 = str1.repeat(2);
