배열 메서드 : join(), push(), pop()

배열은 리스트와 비슷한 객체로서 순회와 변형 작업을 수행하는 메서드를 갖습니다. 배열은 길이, 요소의 자료형이 고정되어있지 않기 때문에, 길이가 언제든지 늘어나거나 줄어들 수 있고, 데이터를 연속적이지 않은 곳에 저장할 수 있습니다. 이러한 배열을 편리하게 다루기 위해 사용되는 메서드에 대하여 알아보겠습니다.

join() 메서드

join()은 배열 요소들을 문자열로 결합해 주어, 문자열로 반환하는 메서드입니다. 

const arrNum = [100, 200, 300, 400, 500];
const text1 = arrNum.join('')
const text2 = arrNum.join(' ')
const text3 = arrNum.join('★')
const text4 = arrNum.join('-')

document.write(text1)
document.write(text2)
document.write(text3)
document.write(text4)
결과보기
100200300400500
100 200 300 400 500
100★200★300★400★500
100-200-300-400-500

push() 메서드

push()는 배열의 끝에 하나 이상의 요소를 추가하고, 배열의 새로운 길이값을 숫자로 반환하는 메서드입니다. 

const arrNum = [100, 200, 300, 400, 500];
const arrPush = arrNum.push(600);  // 마지막 요소에 600을 추가

document.write(arrPush)  // 배열의 길이값(숫자) 출력
document.write(arrNum)  // 배열 요소들 출력
결과보기
6
100, 200, 300, 400, 500, 600

pop() 메서드

pop()은 배열의 마지막 요소를 삭제하고, 삭제된 요소를 반환합니다. 

const arrNum = [100, 200, 300, 400, 500];
const arrPop = arrNum.pop();  // 마지막 요소를 삭제

document.write(arrPop);  // 삭제된 요소 출력
document.write(arrNum);  // 나머지 요소들 출력
결과보기
500
100,200,300,400

메서드 정리

각 메서드들을 정리하면 다음과 같습니다.

메서드 수행 작업 반환값
join() 배열 요소 문자열 결합 문자열
push() 배열 마지막 요소에 추가 숫자(길이값)
pop() 배열 마지막 요소를 삭제 삭제된 요소
