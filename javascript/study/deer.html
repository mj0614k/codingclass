일러스트레이터 : 사슴 그림 그리기

오늘은 일러스트레이터를 활용하여 사슴 그림을 그려보도록 하겠습니다.

스케치 작업

다음과 같은 사슴 그림을 그려볼 예정입니다.

일러스트레이터_02_study_01

이미지에 투명도를 설정해 주고, 펜 툴(Pen tool, 단축키 P)로 선을 그어주며 스케치 작업을 먼저 해 주었습니다. 선 두께는 우선 1px로 작업하고, Shift + W을 사용하여 세세하게 조절해 주었습니다.

일러스트레이터 과정샷

토 나오는 과정입니다. 누구든 저의 노고를 알아주시면 좋겠습니다.

채색 작업

스케치 작업을 마무리했다면, 채색을 해줍니다. 채색 작업을 하기 전, 메뉴의 Object에서 Expand Appearance 작업을 먼저 해 줍니다. 이후, 메뉴의 Object에서 Live Paint를 만들어 줍니다. Live Paint를 사용하면 단축키 K와 Alt(스포이트)를 활용하여 손쉽게 채색 작업을 할 수 있습니다. 다음과 같이 채색을 마쳤습니다.

채색 사진 때려넣어 color2임
