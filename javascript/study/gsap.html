자바스크립트 : GSAP란?

GSAP는 GreenSock에서 만든 자바스크립트 애니메이션 라이브러리입니다. 일반적으로 애니메이션 효과는 CSS나 JQUERY로 구현하는 것이 일반적이지만, 그 이상의 복잡한 애니메이션을 보다 간단하게 구현하기 위해 탄생한 것이 바로 GSAP입니다. 오늘은 GSAP의 사용법을 간단하게 알아보도록 하겠습니다.

1. CDN에서 라이브러리 가져오기

CDN이란 Contents Delivery Network의 약자로, 데이터를 분산된 서버에서 받아오는 것을 말합니다. GSAP이나 jQuery의 라이브러리를 호스팅된 서버에 직접 설치해서 사용할 수도 있지만, CDN을 사용하면 클라이언트가 직접 자신의 위치로 라이브러리를 전송받게 됩니다.

<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.11.0/gsap.min.js"></script>

위와 같이, 라이브러리를 사용하려면 js파일의 링크를 가져오는 것처럼 CDN 서버의 절대 주소를 입력하면 됩니다.

2. GSAP의 객체를 만드는 기본 메서드

gsap.to()
gsap.from()
gsap.fromTo()

gsap의 메서드 to는 첫 번째 인자로 DOM 요소를 받습니다. 마치 자바스크립트의 querySelector와 같이 클래스를 표기하면 됩니다. 두 번째 인자로 옵션값들을 담은 객체를 전달하게 됩니다. 각 메서드들의 사용 방법은 같으며, to는 어디로, from은 어디에서, fromTo는 어디에서 어디로 어떻게 애니메이션을 만들지 결정하기 위한 메서드라고 생각하면 됩니다.
