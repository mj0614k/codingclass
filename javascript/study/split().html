문자열 변경 메서드 : split()

split() 메서드는 문자열에서 원하는 값을 추출하여 배열로 반환합니다.

split() 메서드

split() 메서드는 문자열에서 구분자가 등장하면, 해당 부분을 삭제하고 남은 문자열을 배열로 반환합니다. 구분자가 등장하지 않거나 생략되었을 경우, 배열은 원본 문자열을 유일한 원소로 가집니다.

1. "문자열".split(구분자)

const str1= "javascript reference"
const currentStr1 = str1.split('');  
const currentStr2 = str1.split(' ');
const currentStr3 = str1.split('j');
const currentStr4 = str1.split('a');
const currentStr5 = str1.split('e');
결과보기
['j', 'a', 'v', 'a', 's', 'c', 'r', 'i', 'p', 't', ' ', 'r', 'e', 'f', 'e', 'r', 'e', 'n', 'c', 'e']
['javascript', 'reference']
['', 'avascript reference']
['j', 'v', 'script reference']
['javascript r', 'f', 'r', 'nc', '']

2. "문자열".split(구분자, 갯수);

const str1= "javascript reference";
const currentStr6 = str1.split('', 1);
const currentStr7 = str1.split('', 2);
const currentStr8 = str1.split(' ', 1);
const currentStr9 = str1.split(' ', 2);
결과보기
['j']
['j', 'a']
['javascript']
['javascript', 'reference']

3. "문자열".split(정규식 표현);

const str2 = "java/script/refer/ence";
const currentStr10 = str2.split('/');

const str3 = "java&script&refer!ence";
const currentStr11 = str3.split('!'); // ! 기호만 찾기
const currentStr12 = str3.split('&'); // & 기호만 찾기
const currentStr13 = str3.split(/&|\!/); // | : & 기호와 ! 둘다 찾기

// join() 메서드를 활용한 응용
const str4 = "javascript reference"
const currentStr14 = str4.split('').join();
const currentStr15 = str4.split('').join('★');
const currentStr16 = str4.split('').reverse().join();
const currentStr17 = str4.split('').reverse().join('★');
결과보기
['java', 'script', 'refer', 'ence']
['java&script&refer', 'ence']
['java', 'script', 'refer!ence']
['java', 'script', 'refer', 'ence']
