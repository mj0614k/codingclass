문자열 변경 메서드 : trim() / toUpperCase() / toLowerCase()

trim() 메서드는 문자열의 공백을 제거해 주고, toUpperCase() / toLowerCase() 메서드는 문자열을 대 / 소문자로 변환해 줍니다. 하나씩 살펴보도록 하겠습니다.

trim() 메서드

trim() 메서드는 문자열 양 끝의 공백을 제거합니다. 공백이란 모든 공백문자(space, tab, NBSP 등)와 모든 개행문자(LF, CR 등)를 의미합니다. trimStart() 메서드를 사용하면 앞 공백, trimEnd() 메서드를 사용하면 뒤 공백만을 제거해줄 수 있습니다. 문자열 중간의 공백은 제거할 수 없습니다. 

const str1 = "        javascript        "
const currentStr1 = str1.trim();  // 앞, 뒤 공백 제거
document.write(currentStr1);

const str2 = "        javascript        "
const currentStr2 = str2.trimStart();  // 앞 공백 제거
document.write(currentStr2);

const str3 = "        javascript        "
const currentStr3 = str3.trimEnd();  // 뒤 공백 제거
document.write(currentStr3);
결과보기
javascript
javascript________
________javascript

toUpperCase() 메서드

toUpperCase() 메서드는 문자열을 대문자로 변환해 반환합니다. 


const str1 = "javascript";
const currentStr1 = str1.toUpperCase();
document.write(currentStr1);

const str2 = "jquery";
const currentStr2 = str2.toUpperCase();
document.write(currentStr2);
결과보기
JAVASCRIPT
JQUERY

toLowerCase() 메서드

toLowerCase() 메서드는 문자열을 소문자로 변환해 반환합니다. 

const str1 = "JAVASCRIPT";
const currentStr1 = str1.toLowerCase();
document.write(currentStr1);

const str2 = "JQUERY";
const currentStr2 = str2.toLowerCase();
document.write(currentStr2);
결과보기
javascript
jquery
