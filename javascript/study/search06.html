자바스크립트 : 검색 효과 : sort(), reverse()

2. 속성 출력하기

const cssList = document.querySelector(".search__list .line");
const updateList = () => {
    let listCSS = "";
    cssProperty.forEach((data, index) => {
        listCSS += `${data.num}. ${data.name} : ${data.desc}`;
    });
    cssList.innerHTML = listCSS;
}
updateList();

listCSS라는 빈 문자열 안에 forEach를 사용하여 작성했던 배열 속 각 속성의 번호, 이름, 설명을 추가해 준 뒤, innerHTML로 출력해 주었습니다.

3. 반대로 정렬하기

document.querySelector(".btn1").addEventListener("click", (e) => {
    e.preventDefault();
    cssProperty.reverse();
    updateList();
});

btn1은 '반대로 정렬하기' 버튼을 의미합니다. 간단하게, reverse() 메서드를 사용해주면 됩니다.

4. 오름차순, 내림차순 정렬하기

document.querySelector(".btn2").addEventListener("click", (e) => {
    e.preventDefault();
    cssProperty.sort((a,b) => {
        return a.num - b.num;
    });
    updateList();
});
document.querySelector(".btn3").addEventListener("click", (e) => {
    e.preventDefault();
    cssProperty.sort((a,b) => {
        return b.num - a.num;
    });
    updateList();
});

btn2는 '내림차순 정렬하기', btn3은 '오름차순 정렬하기' 버튼을 의미합니다. 위와 같이 sort() 메서드를 사용하여 간단하게 작업할 수 있습니다.

5. 알파벳순 정렬하기

document.querySelector(".btn4").addEventListener("click", (e) => {
    e.preventDefault();
    cssProperty.sort((a, b) => {
        let x = a.name;
        let y = b.name;
        return x.localeCompare(y);
    });
    updateList();
});
document.querySelector(".btn5").addEventListener("click", (e) => {
    e.preventDefault();
    cssProperty.sort((a, b) => {
        let x = a.name;
        let y = b.name;
        return y.localeCompare(x);
    });
    updateList();
});

btn4는 '알파벳순(a-z) 정렬하기', btn5는 '알파벳순(z-a) 정렬하기' 버튼을 의미합니다. 알파벳 정렬의 경우, 문자열을 비교하는 localeCompare() 메서드를 사용하여 위와 같이 작업할 수 있습니다.

6. 랜덤 정렬하기

document.querySelector(".btn6").addEventListener("click", (e) => {
    e.preventDefault();
    for(let i=cssProperty.length-1; i>=0; i--){
        const randomIndex = Math.floor(Math.random() * (i+1));
        [cssProperty[i], cssProperty[randomIndex]] = [cssProperty[randomIndex], cssProperty[i]];
        cssProperty.innerHTML += cssProperty[i]
    }
    updateList();
});

마지막으로, btn6은 '랜덤 정렬하기' 버튼을 의미합니다. 위 방법은 피셔 예이츠 알고리즘을 이용한 방법입니다.
Math.random() 메서드를 사용하면 0에서 1 사이의 난수를 무작위로 출력합니다. 우리는 정수의 값이 필요하기 때문에, cssProperty의 길이만큼 곱해준 뒤, 소수점을 제거해 줍니다.
그러나 randomIndex를 그대로 출력하면 중복된 값이 발생하게 됩니다. 따라서 객체구조분해할당을 이용해 중복된 값을 제거해 줍니다. 모르겠다면 그냥 외우면 됩니다.
