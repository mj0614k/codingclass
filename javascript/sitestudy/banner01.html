레이아웃 : 배너 유형01

다음과 같은 배너 유형의 레이아웃을 작업해 보도록 하겠습니다.

HTML 소스

배너 유형은 아주 간단합니다. banner__inner 클래스를 가진 부모 div 박스에 백그라운드로 이미지를 삽입하고, 제목과 하위 메시지를 작성하면 완성입니다. 

<section id="bannerType" class="banner__wrap nexon section">
    <h2 class="blind">배너 영역</h2>
    <div class="banner__inner">
        <h3 class="title">귀여운 댕댕</h3>
        <p class="desc">
            더 다양한 댕댕이는 유튜브로 보세요.
            <a href="#" title="유튜브 페이지로 이동">yotube.com</a>
        </p>
        <span class="small">배너 유형01</span>
    </div>
</section>
