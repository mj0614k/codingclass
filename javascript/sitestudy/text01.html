CSS 소스

폰트는 넥슨 Lv1 고딕을 사용하였고, display: flex와 flex-wrap을 이용하여 카드들을 가로로 배치하였습니다. 아이콘은 백그라운드 형식으로 작업해 주었고, 아이콘에 담긴 의미는 특별히 없기 때문에 ir효과는 생략했습니다. 각각의 카드들에 마우스를 대면 배경 색상이 바뀌도록 hover 효과도 작업해 주었습니다.
