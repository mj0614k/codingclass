HTML 소스

카드 유형과 유사하게, 두 개의 카드를 만들어 이미지를 백그라운드 방식으로 삽입했습니다. 웹 표준 준수를 위해, a 태그에는 title을 추가해 주었습니다. 

<section id="imageType01" class="card__wrap nexon section">
    <h2>세상의 귀여운 댕댕이</h2>
    <p>강아지 종류는 매우 다양합니다. 하지만 하나같이 정말 귀엽습니다.</p>
    <div class="image__inner container">
        <div class="image img1">
            <h3 class="image__title">세상의 귀여운 댕댕이</h3>
            <p class="image__desc">누가 뭐래도 댕댕이는 참 귀엽다.<br>
                왜 나는 댕댕이가 없을까?</p>
            <a class="image__btn" href="#" title="자세히 보기">
                <div class="more">자세히 보기</div>
            </a>
        </div>
        <div class="image img2">
            <h3 class="image__title">세상의 귀여운 댕댕이</h3>
            <p class="image__desc">누가 뭐래도 댕댕이는 참 귀엽다.<br>
                왜 나는 댕댕이가 없을까?</p>
            <a class="image__btn blue" href="#" title="자세히 보기">
                <div class="more">자세히 보기</div>
            </a>
        </div>
    </div>
</section>

CSS 소스

폰트는 넥슨 Lv1 고딕을 사용하였고, display: flex를 이용하여 두 개의 이미지를 가로로 배치하였습니다.

@import url('https://webfontworld.github.io/NexonLv1Gothic/NexonLv1Gothic.css');

    font-family: 'NexonLv1Gothic';
    font-weight: 400;
}
