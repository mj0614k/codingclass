CSS 소스

각각의 목록은 display: block으로 블록 구조로 만들어 주고, display: flex를 이용하여 배치해 주었습니다. 

/* 배경 */
.gray {
    background-color: #f5f5f5;
}

/* footerType */
.footer__menu {
    display: flex;
    margin-bottom: 70px;
}
.footer__menu > div {
    width: 16%;
}
.footer__menu div h3 {
    font-size: 18px;
    margin-bottom: 20px;
}
.footer__menu div li a {
    margin-bottom: 10px;
    display: block;
    font-size: 14px;
    color: #666;
}
.footer__right {
    border-top: 1px solid #d9d9d9;
    text-align: center;
    padding-top: 40px;
    color: #666;
    line-height: 1.5;
}
