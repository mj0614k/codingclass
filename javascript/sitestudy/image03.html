CSS 소스

폰트는 고운 바탕을 사용하였고, 이번에는 flex가 아닌 grid를 통해 레이아웃을 작업해 주었습니다. 각각의 아이콘에는 웹 표준성을 준수할 수 있도록 ir효과를 넣어주었습니다. 아이콘을 삽입할 때는 이미지 스프라이트 기법을 사용하였는데요, 게시글 상단의 피그마에서 아이콘들의 집합 프레임을 확인할 수 있습니다. 해당 프레임을 svg 파일로 저장한 뒤, background-position으로 이미지의 위치를 조정시켜 손쉽게 이미지를 삽입하였습니다.
