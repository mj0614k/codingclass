레이아웃 : 헤더 유형01

다음과 같은 헤더 유형의 레이아웃을 작업해 보도록 하겠습니다.

HTML 소스

헤더 유형은 header__wrap 안에 header__inner 박스를 만들어 주고, logo, menu, member의 세 부분으로 나누어 주었습니다. logo에는 웹사이트의 제목(로고), menu에는 다섯 가지의 메뉴, member에는 로그인 버튼을 만들어 주었습니다. 

<section id="headerType" class="header__wrap nexon">
    <div class="header__inner">
        <div class="header__logo">
            <a href="#">web <em>site</em></a>
        </div>
        <nav class="header__menu">
            <ul>
                <li><a href="#">헤더 영역</a></li>
                <li><a href="#">슬라이드 영역</a></li>
                <li><a href="#">배너 영역</a></li>
                <li><a href="#">콘텐츠 영역</a></li>
                <li><a href="#">푸터 영역</a></li>
            </ul>
        </nav>
        <div class="header__member">
            <a href="#">로그인</a>
        </div>
    </div>
</section>

CSS 소스

각각의 위치는 flex, space-between 속성을 이용해 설정해 주었습니다. 각각의 메뉴 버튼, 로그인 버튼에는 마우스를 대면 background 색상이 점차적으로 변화하는 hover 효과를 추가해 주었습니다. 

/* headerType */
.header__inner {
    width: 100%;
    height: 70px;
    display: flex;
    align-items: center;
    justify-content: space-between;
    padding: 0 20px;
    box-sizing: border-box;
    border-bottom: 1px solid #ccc;
}
.header__logo {
    width: 20%;
    font-size: 30px;
    font-weight: 700;
    text-transform: uppercase;
}
.header__logo em {
    font-size: 18px;
    font-weight: 400;
}
.header__menu {
    width: 60%;
    text-align: center;
}
.header__menu li {
    display: inline;
}
.header__menu li a {
    padding: 13px 30px;
    margin: 0 5px;
    transition: background-color 0.3s;
}
.header__menu li a:hover {
    background-color: #f1f1f1;
    border-radius: 5px;
}
.header__member {
    width: 20%;
    text-align: right;
}
.header__member a {
    font-size: 16px;
    border: 1px solid #000;
    padding: 10px 30px;
    border-radius: 50px;
    transition: all 0.3s;
}
.header__member a:hover {
    background-color: #000;
    color: #fff;
}
