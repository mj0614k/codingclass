CSS 소스

폰트는 에스코어 드림을 사용했고, 컨테이너 안에 3개의 카드를 만들어 display:flex로 가로 정렬을 해 주었습니다. blind 효과는 ir 효과와 마찬가지로, 글자가 눈에 보이지 않게 작업하지만, 스크린 리더기에서는 읽히게끔 만들어 웹 접근성을 준수하게끔 합니다.

HTML 소스

각각의 카드는 header, body, footer 세 가지 영역으로 나누어 작업하였습니다. footer 영역도 display:flex를 사용하여 정렬을 해 주었습니다. figcaption 태그는 figure 태그 안에서, 다른 콘텐츠에 대한 설명 혹은 범례를 나타냅니다.
