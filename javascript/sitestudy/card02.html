CSS 소스

ir 효과는 글자를 시각적으로 보이지 않게 작업해주고 있지만, 스크린 리더기에서 글자가 읽히게 해 줌으로써 웹 표준을 준수할 수 있습니다.

/* fonts */
@import url('https://webfontworld.github.io/gmarket/GmarketSans.css');

.gmark {
    font-family: 'GmarketSans';
    font-weight: 500;
}

/* reset */

    margin: 0;
    padding: 0;
}

    text-decoration: none;
    color: #000;
}

/* common */

}

.section {
    padding: 120px 0;
}

    line-height: 1;
    text-align: center;

}

    font-size: 22px;
    font-weight: 300;
    color: #666;

    margin-bottom: 70px;
}

.gray {
    background-color: #f5f5f5;
}

/* ir 효과(이미지 대체 효과 == alt) */

    overflow: hidden;

    line-height: 0;
    text-indent: -9999px;
}

}
