다음과 같은 슬라이드 유형의 레이아웃을 작업해 보도록 하겠습니다. 슬라이드 유형은 버튼을 클릭하면 다음 이미지로 넘어가는 레이아웃 유형입니다. 오늘은 스크립트 구현은 하지 않고, 레이아웃 과정만 작업하도록 하겠습니다.

CSS 소스

화살표에 대한 설명인 '이전 이미지', '다음 이미지' 텍스트는 ir 효과를 주어 숨김 처리해 주었습니다. blind 효과와 ir 효과는 웹 표준성 준수를 위해 사용한다는 점에서 공통점이 있지만, blind 효과는 이미지 없이 단순히 글자를 숨기는 것이고, ir 효과는 이미지와 함께 사용하여 이미지의 alt 효과를 대체하는 효과라는 점에서 차이가 있습니다. 

/* 이미지 대체(ir) 효과 */
.ir {
    display: block;
    overflow: hidden;
    font-size: 0;
    line-height: 0;
    text-indent: -9999px;
}
