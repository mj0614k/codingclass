프로그래머스 1단계 : 약수의 합

코딩테스트 연습 > 연습문제 > 약수의 합

💩 문제 설명

정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.

💩 제한 조건

☝️ n은 0 이상 3000이하인 정수입니다.

💩 문제 풀이

function solution(n) {
    var answer = 0;
    for(i = 1; i<=n; i++){
        if(n % i ==0) answer += i; 
    }
    return answer;
}

1부터 n까지 증가하는 변수 i를 for문 안에서 선언하고, n을 i로 나누었을 때, 나누어 떨어지는 수들(약수들)을 더하도록 작업해 주었습니다.

💩 다른 풀이

function solution(n, a=0, b=0) {
    return n<=a/2?b:solution(n,a+1,b+=n%a?0:a);
}

다른 풀이들도 다 거기서 거기길래 특이하게 생긴 녀석을 데리고 와 봤습니다. 재귀함수에 삼항연산자를 중복으로 사용했다는데 무슨 소리인지는 모르겠네요. 제가 이걸 이해할 수 있는 날이 오겠죠?
