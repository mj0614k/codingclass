프로그래머스 1단계 : 문자열 내 p와 y의 개수

코딩테스트 연습 > 연습문제 > 문자열 내 p와 y의 개수

💩 문제 설명

대문자와 소문자가 섞여있는 문자열 s가 주어집니다. s에 'p'의 개수와 'y'의 개수를 비교해 같으면 True, 다르면 False를 return 하는 solution를 완성하세요. 'p', 'y' 모두 하나도 없는 경우는 항상 True를 리턴합니다. 단, 개수를 비교할 때 대문자와 소문자는 구별하지 않습니다. 예를 들어 s가 "pPoooyY"면 true를 return하고 "Pyy"라면 false를 return합니다.

💩 제한 조건

☝️ 문자열 s의 길이 : 50 이하의 자연수
✌️ 문자열 s는 알파벳으로만 이루어져 있습니다.

💩 문제 풀이

function solution(s){
    let answer;
    p = s.toLowerCase().match(/p/g);
    y = s.toLowerCase().match(/y/g);
    if(p&&y) {
        if(p.length == y.length) {
            answer = true
        } else {
            answer = false
        }
    } else if (p||y){
        answer = false;
    } else {
        answer = true;
    }
    return answer;
}

match() 메서드로 p와 y를 각각의 개수만큼 원소로 가지는 배열을 만들고, 그 길이를 비교해 줍니다. 모든 경우의 수를 작성해 주었습니다. 저보다 더럽게 푼 사람 있으면 나와 보세요.

💩 다른 풀이

function numPY(s){
    return s.toUpperCase().split("P").length === s.toUpperCase().split("Y").length;
}

이해가 될 듯 말 듯 해서 실험해 봤는데, 좌변과 우변의 값은 각각 p의 개수 + 1, y의 개수 + 1이 출력됩니다. split() 메서드는 생각도 못 했네요.
