프로그래머스 1단계 : 정수 제곱근 판별

코딩테스트 연습 > 연습문제 > 정수 제곱근 판별

💩 문제 설명

임의의 양의 정수 n에 대해, n이 어떤 양의 정수 x의 제곱인지 아닌지 판단하려 합니다. n이 양의 정수 x의 제곱이라면 x+1의 제곱을 리턴하고, n이 양의 정수 x의 제곱이 아니라면 -1을 리턴하는 함수를 완성하세요.

💩 제한 조건

☝️ n은 1이상, 50000000000000 이하인 양의 정수입니다.

💩 문제 풀이

function solution(n) {
    let answer;
    for(x=1; x<=n; x++){
        if(n == x * x){
            answer = (x + 1) * (x + 1);
            break;
        } else { answer = -1 }
    }
    return answer;
}

for문을 사용하여, n이 x의 제곱이면 ( x+1 )의 제곱을 answer에 입력하고, 아니라면 answer에 -1을 입력해 주었습니다.

💩 다른 풀이

function nextSqaure(n){
    switch(n % Math.sqrt(n)){
      case 0:
        return Math.pow(Math.sqrt(n) + 1, 2);
      default:
        return -1;
    }
}

Math.sqrt()는 제곱근을 리턴하는 메서드라고 합니다. Math.pow()는 (a, b)의 값이 주어졌을 때, a의 b제곱을 리턴하는 메서드라고 합니다. 역시 머리가 좋으면 손가락이 덜 힘드네요. 전 아직은 손가락이 아주 바쁩니다.
