프로그래머스 1단계 : 가운데 글자 가져오기

코딩테스트 연습 > 연습문제 > 가운데 글자 가져오기

💩 문제 설명

단어 s의 가운데 글자를 반환하는 함수, solution을 만들어 보세요. 단어의 길이가 짝수라면 가운데 두글자를 반환하면 됩니다.

💩 제한 조건

☝️ s는 길이가 1 이상, 100이하인 스트링입니다.

💩 문제 풀이

function solution(s) {
    let x;
    (s.length % 2 == 0) ? x = s.length / 2 : x = s.length / 2 - 0.5;
    return (s.length % 2 == 0) ? s.charAt(x - 1) + s.charAt(x) : s.charAt(x);
}

변수 x를 선언해 주고, s의 길이가 홀수인지, 짝수인지에 따라 x에 중간 인덱스를 입력해 주었습니다. (s.length % 2 == 0)라고 똑같은 식을 두 번 쓴 게 마음에 안 드는데, 지금 생각해 보니 Math 메서드를 써서 한번에 작업할 걸 그랬습니다.

💩 다른 풀이

function solution(s) {
    return s.substr(Math.ceil(s.length / 2) - 1, s.length % 2 === 0 ? 2 : 1);
}

Math.ceil에 이어 substr() 메서드를 사용하여 더 간단하게 작업할 수 있네요!
