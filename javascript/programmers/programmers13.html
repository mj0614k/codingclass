프로그래머스 1단계 : x만큼 간격이 있는 n개의 숫자

코딩테스트 연습 > 연습문제 > x만큼 간격이 있는 n개의 숫자

💩 문제 설명

함수 solution은 정수 x와 자연수 n을 입력 받아, x부터 시작해 x씩 증가하는 숫자를 n개 지니는 리스트를 리턴해야 합니다. 다음 제한 조건을 보고, 조건을 만족하는 함수, solution을 완성해주세요.

💩 제한 조건

☝️ x는 -10000000 이상, 10000000 이하인 정수입니다.
✌️ n은 1000 이하인 자연수입니다.

💩 문제 풀이

function solution(x, n) {
    var answer = [];
    for(i=1; i<=n; i++){
        answer.push(x * i)
    }
    return answer;
}

for문으로 n의 개수만큼의 길이를 가진 배열을 만들어 주었습니다. 아주 간단해서 더 쓸 말이 없네요.

💩 다른 풀이

function solution(x, n) {
    return Array(n).fill(x).map((v, i) => (i + 1) * v)
}

여기에서 Array(n)은 n개의 길이를 가진 배열을 만듭니다. fill(x)는 그 배열의 요소를 x로 채웁니다. 이후, map() 메서드를 사용하여 각각의 요소에 index + 1에 해당하는 값을 더해주면 됩니다. 천재적인 발상!
